# Installing SparseML

SparseML is Neural Magic's core toolkit for streamlining the optimization of deep learning models using cutting-edge sparsification methods (e.g., pruning, quantization).
This guide covers installation options, including PyPI, GitHub, and specialized applications for various AI domains.

## Prerequisites

### Hardware Requirements

- <b>CPU:</b> Intel, AMD, or ARM
- <b>RAM:</b> 4GB+ recommended (depends on your models and workflows)
- <b>GPU:</b> NVIDIA (recommended, 16GB+ VRAM); AMD (limited support)

### Software Requirements

- <b>OS:</b> Linux (e.g., Ubuntu, CentOS, Red Hat); MacOS (limited support)
- <b>Python:</b> 3.8 - 3.11
- <b>PyTorch:</b> >= 1.1.0, < 2.2


## Installation

### PyPI

For the most common use case, install the current release version of SparseML using PyPi:

<VersionInjector targetProduct="sparseml" targetVersion="==VERSION" prepend="==">
    ```bash
    pip install sparseml==VERSION
    ```
</VersionInjector>

### GitHub (Advanced)

For development purposes or advanced use cases, install directly from the GitHub repository:

<VersionInjector targetProduct="sparseml" prepend="v" currentTag="main" ignoreNightly={true} ignoreLastVersion={true}>
    ```bash
    pip install git+https://github.com/neuralmagic/sparseml.git@VERSION
    ```
</VersionInjector>

Or from a locally cloned repository:

<VersionInjector targetProduct="sparseml" prepend="tags/v" currentTag="main" ignoreNightly={true} ignoreLastVersion={true}>
    ```bash
    git clone https://github.com/neuralmagic/sparseml.git
    cd sparseml
    git checkout VERSION
    pip install -e .[dev]
    ```
</VersionInjector>
