# Installing SparseZoo

SparseZoo is Neural Magic's repository of pre-sparsified deep learning models across various domains, such as computer vision and NLP.
The easiest way to access and use these models is through:
- <b>SparseML:</b> For model optimization, integration into your training pipelines
- <b>DeepSparse:</b> For high-performance deployment of sparsified models

## Prerequisites

### Hardware Requirements

- <b>CPU:</b> Intel, AMD, or ARM

### Software Requirements

- <b>OS:</b> Linux (e.g., Ubuntu, CentOS, Red Hat); MacOS (limited support)
- <b>Python:</b> 3.8 - 3.11

## Installation

### PyPI

For the most common use case, install the current release version of SparseZoo using PyPi:

<VersionInjector targetProduct="sparsezoo" targetVersion="==VERSION" prepend="==">
    ```bash
    pip install sparsezoo==VERSION
    ```
</VersionInjector>

### GitHub (Advanced)

For development purposes or advanced use cases, install directly from the GitHub repository:

<VersionInjector targetProduct="sparsezoo" prepend="v" currentTag="main" ignoreNightly={true} ignoreLastVersion={true}>
    ```bash
    pip install git+https://github.com/neuralmagic/sparsezoo.git@VERSION
    ```
</VersionInjector>

Or from a locally cloned repository:

<VersionInjector targetProduct="sparsezoo" prepend="tags/v" currentTag="main" ignoreNightly={true} ignoreLastVersion={true}>
    ```bash
    git clone https://github.com/neuralmagic/sparsezoo.git
    cd sparsezoo
    git checkout VERSION
    pip install -e .[dev]
    ```
</VersionInjector>
